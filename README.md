# Bluetooth Arduino Car 🚗

This project shows how to build a Bluetooth-controlled Arduino car.

## 🔧 Components

* Arduino Uno
* HC-05 Bluetooth Module
* L298N Motor Driver
* DC Motors
* 18650 Battery

## 📂 Files

* Arduino Code (.ino)
* Circuit Diagram

## 📱 Control

Use any Bluetooth controller app from Play Store.
